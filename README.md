# SmallDownloader
简单，小巧，可靠，灵活的Android下载引擎! Simple,Small, Reliable And Flexible Android download engine

# 优点
1、只关心核心的线程调度，数据保存可以自由定制<br/>
2、网络下载模块也可以非常方便的定制，便于和现在流行的网络库okhttp，防劫持的库httpdns等结合<br/>
3、该库里面所有的线程一旦空闲超时都会自动销毁，释放资源<br/>
